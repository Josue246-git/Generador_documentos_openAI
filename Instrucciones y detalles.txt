Para levantar la aplicación se debe crear dos instancias de consola en donde:

1. Ingresar primero a la carpeta ./client y ejecutar el servidor frontend con npm run dev
2. En la otra instancia ingresar a la carpeta ./server y ejecutar el servidor frontend con npm start 

Nota: si no esta instalado los modulos necesarios se deberá ejecutar el comando npm install tanto en la instancia del frontend como del backend y después ejecutarlos con los comandos anteriormente mencionados


TECONOLGIAS USADAS:

FRONTEND:
- React por medio del empaquetador VITE
- TailwindCss
- Html y JavaScript 

BACKEND:
-JavaScript
-Node js con Express

BD:
- PostgreSQL

OTROS:
